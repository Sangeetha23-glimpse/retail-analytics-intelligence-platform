# 🛍️ Retail Analytics Intelligence Platform

---

## 📌 Overview

This is a complete **end-to-end Retail Analytics Platform** built using SQL, Python, Alteryx, and dashboarding tools.
The project analyzes **9,800 transactions ($2.26M revenue)** from 2015–2018 to generate business insights, customer segmentation, and sales forecasting.

---

## 🚀 Tech Stack

* **SQL (MySQL)** → Data warehouse & KPI queries
* **Python** → EDA, RFM segmentation, forecasting (Prophet), ML
* **Alteryx** → ETL pipeline
* **Power BI / HTML** → Interactive dashboards

---

## 📊 Key Features

### 🔍 Exploratory Data Analysis (EDA)

* Revenue trends & seasonality analysis
* Regional and category performance
* Delivery delay analysis

---

### 🧠 Customer Segmentation (RFM)

* Champions = **29.4% of total revenue**
* Identified **at-risk customers** for retention strategy

---

### 📈 Sales Forecasting

* Model: Prophet (time-series forecasting)
* Accuracy: **MAPE = 16.8%**
* Forecast: **$311K for Jan–Jun 2019**

---

### 🤖 Delivery Delay Prediction

* Machine Learning model (Gradient Boosting)
* Identified root cause of delays
* Found **Standard Class responsible for delays**
